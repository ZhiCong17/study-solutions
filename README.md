Tuition Centre Landing Page

A responsive single-page landing website built for study solutions. The website showcases the centre's contact information, services, course offerings, testimonials, contact form and location.

Features

- Responsive design for desktop, tablet, and mobile
- Hero section with call-to-action
- Courses/Programs section
- Student/Parent testimonials
- Contact section
- Modern UI built with React and Vite

Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS

Getting Started

Prerequisites

- Node.js
- npm

Installation
npm install

Run Development Server
npm run dev

Build for Production
npm run build

Preview Production Build
npm run preview
